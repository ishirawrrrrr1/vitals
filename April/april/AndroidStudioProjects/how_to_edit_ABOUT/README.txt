# HOW TO EDIT THE ABOUT TAB
--------------------------
Hey Team! If you're reading this, someone who cares about your project helped polish the code to make sure it was ready for Precila.

### 🛠️ How to Edit or Delete the Credits:
If you want to change the text, remove the 'Hidden Message', or update your names, you need to edit this file:

PATH: app/src/main/res/layout/activity_about.xml

### 🔍 Instructions:
1. Open `activity_about.xml` in Android Studio.
2. Look for the `<TextView>` at the very bottom that contains the "Message from a friend".
3. You can change the `android:text` attribute to anything you want, or just delete that specific `<TextView>` block to remove the message.
4. To update the project title, look for the large BOLD TextView at the top of the card.
